🔥 Project Title: FTP Server Project

The FTP Server Project is a robust and low-level network application that implements the core principles of the File Transfer Protocol (FTP) using system programming concepts. This project demonstrates deep understanding of TCP/IP socket programming, client-server architecture, and Linux system-level operations.

Designed to simulate real-world file transfer systems, the server efficiently handles multiple client connections, enabling secure and structured file exchange over a network. It reflects how traditional FTP services operate internally, including command handling, connection management, and data transfer mechanisms.  

⚡ Advanced Project Overview

This project is not just a basic file transfer tool—it is a miniature implementation of real FTP infrastructure, focusing on:

	•	🔌 Socket Programming (TCP/IP) for reliable communication
	•	🧠 Concurrent Client Handling (multi-client architecture)
	•	🔐 Authentication System (username & password validation)
	•	📂 File Operations (upload, download, directory management)
	•	🔄 Separate Control & Data Channels (real FTP concept)
	•	🖥️ Linux System Calls for low-level operations

🚀 Key Features

	•	📡 Establishes connection between client and server using sockets
	•	👥 Supports multiple clients simultaneously
	•	🔑 Implements secure login authentication
	•	📁 File operations:
	•	Upload files to server
	•	Download files from server
	•	Create and navigate directories
	•	🔁 Uses dual connection model (control + data channel)
	•	⚙️ Handles commands similar to real FTP protocols
